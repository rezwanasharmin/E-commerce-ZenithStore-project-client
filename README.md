# 🛍️ ZenithStore — Client Side Frontend

ZenithStore is a modern, high-performance E-Commerce frontend web application built with **React**, **TypeScript**, **Tailwind CSS**, **Framer Motion**, and **Agentic AI Features**.

---

## 🌟 Key Features

- **🎨 Premium HSL Aesthetics**: Dark/Light mode theme switcher, glassmorphic headers, smooth Framer Motion micro-animations.
- **🛍️ Complete Catalog & Multi-Filter**: Search bar, 3-field multi-filtering (Category, Price Range, Minimum Rating), sorting options, and pagination.
- **🤖 Agentic AI Assistant**: Floating AI shopping concierge (`AIAssistantWidget.tsx`) with quick prompt pills, typing indicator, and context-aware responses.
- **✨ AI Product Summary**: AI-generated product insights (`ProductDetails.tsx`), sentiment analysis (92%+ positive score), pros/cons breakdown, and verdict.
- **📊 Sales Analytics Dashboard**: Integrated Recharts AreaChart & PieChart for real-time sales trends and category distributions.
- **🔐 Full Authentication**: Email/Password login, Register, Auto-fill Demo User button, and Google OAuth social sign-in.
- **📦 Protected Admin Routes**: Add Item (`/items/add`) and Manage Inventory (`/items/manage`) with real-time deletion modals.
- **📱 100% Fully Responsive**: Optimized across Mobile (375px), Tablet (768px), and Desktop (1536px) viewports.

---

## 🛠️ Technology Stack

- **Framework**: React 18 (TypeScript)
- **Bundler**: Vite
- **Styling**: Tailwind CSS v4, Vanilla CSS Design System
- **Animations**: Framer Motion
- **Charts**: Recharts
- **Icons**: Lucide React
- **Icons & Auth**: `@react-oauth/google`
- **HTTP Client**: Axios

## 🚀 Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/rezwanasharmin/E-commerce-ZenithStore-project-client.git
   cd E-commerce-ZenithStore-project-client
   ```

2. **Install Dependencies**:
   ```bash
   npm install
   ```

3. **Start Development Server**:
   ```bash
   npm run dev
   ```

4. **Build Production Bundle**:
   ```bash
   npm run build
   ```
